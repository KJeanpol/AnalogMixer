# SmarCocktail

SmarCocktails is a React App that Controls a Beverage Dispenser hardware by a Microcontroller and MQTT comuncation

## Installation
Download [Expo](https://play.google.com/store/apps/details?id=host.exp.exponent&hl=es_CR) into your Android Smartphone  or [Expo](https://apps.apple.com/es/app/expo-client/id982107779) into your ios Device

First, install [Node.js](https://nodejs.org/es/download/) in your computer. Then download or clone the project into your computer, is recommended to install  [Visual Studio Code](https://code.visualstudio.com/) to manage the project. 

Open the project on Visual Studio Code, and open a terminal on Proyecto1/Front-End/Client-View route  

Then run:

```bash
npm install
expo start
```
A new window in your web browser will open, and a QR will be display, select tunnel option. Open Expo in your smartphone and scan the QR. Finally the app will be display in the smartphone
